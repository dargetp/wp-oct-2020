Wordpress 
à l'origine => blog = web log => journal de bord que l'on met sur internet
écrire ses humeurs 

=> Wordpress a grandi (très utilisé) => CMS

Content Management System (CMS) <=> Système de Gestion de Contenu (SGC)

site internet => découpée en 2 parties 
Front Office => partie visible par les internautes 
lire vos articles / voir les images ...

Back Office  => partie dédiée uniquement pour les gestionnaires du site
uniquement accessible si on a un login et mot de passe correct 
dans cette partie on va gérer tout ce qui va être visible dans le Front Office

rôle => dans le backoffice niveau de droit
Administrateur => gestionnaire qui a tous les droits sur le site
Rédacteur / traducteur / editeur 
Eux aussi ce sont des gestionnaires mais qui ont moins de droits 
= qui ont accès à moins de menu dans le back office
